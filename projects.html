MineSavage Discord Bot

I was approached by a representative from MineSavage to develop a custom and configurable Discord bot to automate some tasks and add additional features to their discord server.

This was an application written in Python - while completing this project, I used JSON files to store data such as data and serialized objects, like channel IDs.

I also used some OOP concepts; by using the Discord Cog Class as a super-class, I was able to load all commands and events on the bot's startup from the /cogs/ directory. My code became more modularised as I could simply iterate over my /cogs/ directory, if the Python file located didn't contain a setup() function then it wouldn't be recognised and subsequently would be ignored.
